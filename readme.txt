== Description ==

Social Lock is like to download plug in ,protecting your contents and premium download link behind social network like boxes.
Your premium link will not be visible until user like , +1 or tweet your post.
Now you can get some more traffic from social network , increase the likes of your facebook pages.

== Installation ==

1. Upload `social-lock.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now with tinyMce button do your magic
